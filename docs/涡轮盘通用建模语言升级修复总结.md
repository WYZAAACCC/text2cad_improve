## 一、问题背景

航空发动机涡轮盘建模测试（`turbine_disc/test_v11/`）暴露了系统级问题：

1. **几何形状完全错误**：盘剖面是纺锤形（中间粗两端细），而非涡轮盘的 hub厚→web薄→rim厚 等强度剖面
2. **榫槽不是枞树状**：`cut_rim_slot_pattern` 只能做折线槽，无圆角、无齿肩语义
3. **系统走上了 primitive 模板路线**：路由层偏好 `deterministic_primitive`，`axisymmetric_turbine_disk` 是 735 行 75 参数的巨型模板

**核心风险**：如果系统走上"全 primitive 模板化"路线，参数化建模已经很成熟，系统的优势将荡然无存。正确方向是**建模语言通用化、原子化、CAD化**——让 LLM 用自由度足够高的通用操作完成任意零件构建。

## 三、修复方案

### 3.1 设计原则

**坚决不走 primitive 模板路线。** 建模语言走"通用化、原子化、CAD化"路线：
- 领域知识 → **verifier**（验证 LLM 建的对不对）
- 几何生成 → **LLM + 通用 CAD op**（LLM 自主建模）

## 五、涡轮盘的正确通用建模链

修复后，LLM 用以下纯通用操作链完成涡轮盘建模：

```
─── 组件 1: disk_body (sketch_profile) ───

Step 1:  create_2d_sketch (plane=XZ)
Step 2:  add_polyline (points=[
           [60,-38],[120,-38],[170,-22],[215,-16],[250,-32],  # 前侧轮廓
           [250, 32],[215, 16],[170, 22],[120, 38],[60, 38]   # 后侧轮廓
         ])
         → R-Z 剖面：hub厚(76mm) → web薄(32mm) → rim厚(64mm)
Step 3:  close_profile
Step 4:  revolve_profile (axis=Z, angle_deg=360)
         → 旋转 360° 生成基础盘体

─── 组件 2: slot_cutter (sketch_profile) ───

Step 5:  create_2d_sketch (plane=XY)
Step 6:  add_polyline / add_arc_segment
         → 枞树槽截面（窄颈→宽齿肩→窄颈→宽齿肩）
Step 7:  fillet_sketch (radius_mm=0.8)
         → 齿根圆角
Step 8:  close_profile
Step 9:  extrude_profile (depth_mm=80, both=True)
         → 沿 Z 方向拉伸成贯穿 rim 的 cutter

─── 组件 3: __assembly__ (composition) ───

Step 10: translate_solid (vector=[250, 0, 0])
         → 将 cutter 移到 rim 外径位置
Step 11: circular_pattern_component (count=60, rotate_copies=True)
         → 圆周阵列 60 个，每个槽径向对齐
Step 12: boolean_cut (target=disk_body, tool=patterned_cutters)
         → 从盘体减去 cutters
```

**关键特征**：全程无任何 `turbine_disk()` 或 `fir_tree_slot()` 专用函数。所谓"涡轮盘"和"枞树槽"只是带语义约束的通用操作组合。
