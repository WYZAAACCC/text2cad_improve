### ⚠️ MINOR — #008 Preflight cost estimator

**Problem:** `BudgetExceeded` is referenced as a raised exception but its type shape is not defined. The agent needs to know whether this is a new exception class, a subclass of `SeekFlowError`, or something else.

**Suggested fix:** Add: `BudgetExceeded` is a new exception class in `seekflow.errors`, subclass of `SeekFlowError`, with fields `limit: float`, `estimated: float`, `model: str`.

### ⚠️ MINOR — #009 Per-tool execution timeout

**Problem:** No clarification on how per-tool timeout interacts with the existing `execution_timeout` parameter on `Agent.run()` (which wraps the entire run in a `ThreadPoolExecutor` with a single timeout). The agent may be confused about whether these timeouts compose, conflict, or are independent.

**Suggested fix:** Add note: "Per-tool timeout is independent of `Agent.run(execution_timeout=...)`. If the tool timeout fires first, the tool returns an error result and the run continues. If the agent-level `execution_timeout` fires first, the entire run is terminated. The tool-level timeout should always be shorter than the agent-level timeout."
