## 背景

开发者需要一种轻量级方式来验证 DeepSeek 工具调用的稳定性：模型是否选择了正确的工具？参数是否准确？最终回答是否符合预期？本 Issue 提供 YAML 驱动的评测框架和 CLI 命令。

## 任务

### 评测数据模型

```python
class ExpectedToolCall(BaseModel):
    name: str
    arguments: dict[str, Any] = {}

class EvalCase(BaseModel):
    id: str
    input: str
    expected_tools: list[ExpectedToolCall] = []
    expected_final_contains: list[str] = []
```

### loader.py — YAML 加载

```python
def load_benchmark(path: str) -> tuple[str, str, list[EvalCase]]
# 返回: (benchmark_name, model, cases)
```

YAML 格式：

```yaml
name: basic_tool_calling
model: deepseek-chat

cases:
  - id: weather_001
    input: "查询杭州天气"
    expected_tools:
      - name: get_weather
        arguments:
          city: "杭州"
    expected_final_contains:
      - "杭州"
```

### runner.py — 执行器

```python
class EvalRunner:
    def __init__(self, runtime: ToolRuntime, model: str)
    def run_file(self, path: str) -> EvalReport
    def run_cases(self, cases: list[EvalCase]) -> EvalReport
```

### metrics.py — 指标计算

6 项指标：

1. `total_cases`
2. `passed_cases`（所有检查通过）
3. `failed_cases`
4. `success_rate`（passed / total）
5. `tool_name_accuracy`（工具名匹配率）
6. `argument_accuracy`（参数匹配率）
7. `final_contains_accuracy`（最终回答包含率）
8. `avg_steps`
9. `avg_latency_ms`

### report.py — 报告输出

```python
class EvalReport(BaseModel):
    name: str
    model: str
    metrics: dict[str, float]
    case_results: list[dict]

    def print(self) -> None  # Rich 格式化输出
```

### cli.py — CLI 命令

使用 Typer + Rich：

```bash
seekflow eval benchmarks/basic_tools.yaml    # 运行评测
seekflow trace view trace.json               # 查看 trace
```

`seekflow eval` 输出示例：

```
Benchmark: basic_tool_calling

Total cases: 20
Passed: 17
Failed: 3
Success rate: 85.00%

Tool name accuracy: 90.00%
Argument accuracy: 82.00%
Final contains accuracy: 88.00%
Average steps: 2.4
Average latency: 1530 ms
```

## 验收标准

- [ ] `load_benchmark("benchmarks/basic_tools.yaml")` 正确解析 YAML
- [ ] `EvalRunner.run_file()` 执行所有 case 并返回 `EvalReport`
- [ ] 工具名正确匹配时 `tool_name_accuracy` 为 100%
- [ ] 参数正确匹配时 `argument_accuracy` 为 100%
- [ ] 最终回答包含预期文本时 `final_contains_accuracy` 正确
- [ ] `EvalReport.print()` 输出格式化报告
- [ ] `seekflow eval benchmarks/basic_tools.yaml` CLI 命令可运行
- [ ] `seekflow trace view trace.json` 使用 Rich 打印事件列表
- [ ] `seekflow --help` 显示所有可用命令
